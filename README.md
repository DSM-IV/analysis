# 해석학 학습자료 (Real Analysis Notes)

🔗 **웹사이트** : https://analysisonline.pages.dev

대학 해석학 강의 내용을 스스로 정리한 개념·예제 증명 웹 자료입니다.
실수 체계부터 챕터별로 정리했고, 각 장마다 본문과 예제 풀이를 나눠 구성했습니다.

## 구성
- **Chapter 1** : 실수 체계 (Real Number System)
- **Chapter 2 ~ 4** : 개념 정리 및 예제 증명
- 각 장별 `chN.html`(개념) / `chN-ex.html`(예제) 페이지
- PDF 증명자료 포함

## 기술
- HTML · MathJax (수식 렌더링)
- Cloudflare Pages 배포

## 만든 이유
수학 복수전공 학습 내용을 직접 웹으로 재구성하며 정리한 개인 프로젝트입니다.
