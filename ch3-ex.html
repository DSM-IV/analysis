EX 3.5 THM 3.4(a)의 서술 정교화 → THM 3.4
한국어

문제. THM 3.4(a)의 서술에서 오류를 찾고 수정하라.

풀이. 본 강의노트 원서(또는 Rudin 3.2(b))의 서술은 "every neighborhood of $p$ contains $p_n$ for all but finitely many $n$"이다. 여기서 주의할 점은 "항(term) $p_n$"과 "치역의 점"을 혼동하면 안 된다는 것이다. 예를 들어 상수 수열 $p_n = p$ ($\forall n$)의 치역은 $\{p\}$이지만 항의 개수는 무한하다. 따라서 정확한 서술은:

수정된 서술. $\{p_n\}$이 $p$에 수렴할 필요충분조건은 $p$의 모든 근방 $V$에 대해 집합 $\{n \in \mathbb{N} : p_n \notin V\}$가 유한 집합인 것이다.

증명은 원래의 THM 3.4(a) 증명과 동일하다: ($\Rightarrow$) $V = N_\varepsilon(p)$, $n \geq N$이면 $p_n \in V$이므로 $\{n : p_n \notin V\} \subset \{1,\ldots,N-1\}$ 유한. ($\Leftarrow$) 유한이면 최대값 $N^*$를 넘는 $n$에 대해 $p_n \in V = N_\varepsilon(p)$, 즉 $d(p_n,p)<\varepsilon$.

□
English

Problem. Find and correct errors in Theorem 3.4(a).

Solution. The issue is confusing "terms $p_n$" with "points in the range". A constant sequence $p_n = p$ has range $\{p\}$ but infinitely many terms. The corrected statement: $\{p_n\}$ converges to $p$ iff for every neighborhood $V$ of $p$, the index set $\{n \in \mathbb{N} : p_n \notin V\}$ is finite. The proof is identical to the original. $\square$

EX 3.8 수렴 수열이면 극한은 극한점인가? → THM 3.6
한국어

문제. $(X,d)$가 거리 공간, $E \subset X$, $p \in X$라 하자. $E$ 안의 수열 $\{p_n\}$이 존재하여 $\lim p_n = p$라 하자. $p$가 $E$의 극한점인가? 증명하거나 반례를 들라.

풀이. 일반적으로 성립하지 않는다.

반례. $X = \mathbb{R}$ (보통 거리), $p = 0$, $E = \{0\}$, $p_n = 0$ ($\forall n$). 그러면 $\{p_n\}$은 $E$ 안의 수열이고 $\lim p_n = 0 = p$. 그러나 $E = \{0\}$은 유한 집합이므로 THM 2.35에 의해 $\mathbb{R}$에서 극한점을 갖지 않는다. 따라서 $p$는 $E$의 극한점이 아니다.

보완. 만약 $p_n \neq p$ ($\forall n$)인 조건이 추가되면 $p$는 $E$의 극한점이 된다 (EX 3.17의 역 방향). 왜냐하면 임의의 $\varepsilon > 0$에 대해 $n$을 잡으면 $p_n \in E$, $p_n \neq p$, $d(p, p_n) < \varepsilon$이기 때문이다.

□
English

Problem. Let $(X,d)$ be a metric space, $E \subset X$, $p \in X$. Assume there exists a sequence $\{p_n\}$ in $E$ with $\lim p_n = p$. Is $p$ a limit point of $E$?

Disprove. Counterexample: $X = \mathbb{R}$, $p = 0$, $E = \{0\}$, $p_n \equiv 0$. Then $\lim p_n = 0 \in E$ but $E$ finite has no limit points (THM 2.35). However, if $p_n \neq p$ for all $n$, then $p$ is a limit point. $\square$

EX 3.10 $\mathbb{N}$의 공이 아닌 부분집합에는 최소원소 존재 → DEF 3.9
한국어

문제. $A \subset \mathbb{N}$, $A \neq \emptyset$이라 하자. $\min A := \min\{a : a \in A\}$가 존재함을 보여라.

풀이. ($\mathbb{N}$의 well-ordering 성질)

$A \neq \emptyset$이므로 $a_0 \in A$를 택한다. 집합 $B := \{k \in \mathbb{N} : k \leq a_0\} \cap A$은 공이 아니고 ($a_0 \in B$), 원소 수가 $a_0$ 이하인 유한 집합이다.

유한 집합에서는 최소원소가 명백히 존재한다 (원소 수에 대한 귀납법, 또는 $B = \{b_1,\ldots,b_m\}$에서 $\min := \min\{b_1, \min\{b_2,\ldots,b_m\}\}$을 재귀적으로 정의). $b^* := \min B$라 하자.

주장: $b^* = \min A$. 임의의 $a \in A$에 대해, $a \leq a_0$이면 $a \in B$이므로 $b^* \leq a$. $a > a_0 \geq b^*$이면 $b^* < a$. 따라서 $b^* \leq a$ ($\forall a \in A$), 즉 $b^* = \min A$.

□
English

Problem. Let $A \subset \mathbb{N}$, $A \neq \emptyset$. Show $\min A$ exists.

Pick $a_0 \in A$. $B := \{k \in \mathbb{N} : k \leq a_0\} \cap A$ is a non-empty finite set and has a minimum $b^*$. For $a \in A$ with $a \leq a_0$, $a \in B$ so $b^* \leq a$; for $a > a_0$, $a > b^*$. Thus $b^* = \min A$. $\square$

EX 3.11 $p$가 무한 번 나타나면 수렴 부분수열 존재 → DEF 3.9
한국어

문제. $(X,d)$가 거리 공간, $\{p_n\}$이 $X$ 안의 수열, $p \in X$라 하자. 무한히 많은 $j \in \mathbb{N}$에 대해 $p = p_j$라 가정하자 (즉, $\{j : p = p_j\}$가 무한). 부분수열 $\{p_{n_i}\}$가 존재하여 $\lim p_{n_i} = p$임을 보여라.

풀이. 집합 $S := \{j \in \mathbb{N} : p = p_j\}$는 무한. EX 3.10의 well-ordering에 의해 최소원소를 반복적으로 골라서 $n_1 < n_2 < n_3 < \cdots$를 얻는다.

구체적으로: $n_1 := \min S$. 귀납적으로 $n_{k+1} := \min\{j \in S : j > n_k\}$. $S$가 무한이므로 $\{j \in S : j > n_k\}$도 공집합이 아니다 (무한 집합에서 유한 개를 뺀 것은 여전히 무한). 따라서 $n_k$들이 모든 $k$에 대해 잘 정의된다.

각 $i$에 대해 $p_{n_i} = p$ (by 정의)이므로 $\lim_{i \to \infty} p_{n_i} = p$ (상수 수열).

□
English

Problem. If $\{j \in \mathbb{N} : p = p_j\}$ is infinite, show there is a subsequence $\{p_{n_i}\}$ of $\{p_n\}$ with $\lim p_{n_i} = p$.

Let $S := \{j : p_j = p\}$. By well-ordering (EX 3.10), define $n_1 = \min S$ and $n_{k+1} = \min\{j \in S : j > n_k\}$; all well-defined since $S$ infinite. Then $p_{n_i} = p$ for all $i$, so $\lim p_{n_i} = p$. $\square$

EX 3.12 무한 부분치역에서 수렴이 있으면 부분수열도 수렴 → DEF 3.9
한국어

문제. $(X,d)$가 거리 공간, $\{p_n\}$이 $X$ 안의 수열, $p \in X$이고 $\{\tilde p_n\}$이 $\{p_n : n \in \mathbb{N}\}$ 안의 수열이라 하자. 즉 $\{\tilde p_n : n \in \mathbb{N}\} \subset \{p_n : n \in \mathbb{N}\}$. $\{\tilde p_n : n \in \mathbb{N}\}$이 무한이고 $\lim \tilde p_n = p$이면, $\{p_n\}$의 부분수열 $\{p_{n_i}\}$가 존재하여 $\lim p_{n_i} = p$임을 보여라.

풀이. $\varepsilon > 0$을 주자. $\lim \tilde p_n = p$이므로 $K$가 존재하여 $k \geq K$이면 $d(\tilde p_k, p) < \varepsilon$.

부분수열을 귀납적으로 구성한다. 각 $\tilde p_k \in \{p_n : n \in \mathbb{N}\}$이므로 $\tilde p_k = p_{m_k}$인 $m_k \in \mathbb{N}$이 존재한다. 집합 $\{\tilde p_n\}$이 무한이므로 $\{m_k\}$도 무한 값을 가진다 (만약 유한이면 치역도 유한).

구체적으로, $n_1 \in \mathbb{N}$을 $p_{n_1} = \tilde p_{k_1}$로 잡는다 ($k_1$은 임의). 다음으로 $k_2 > k_1$을 충분히 크게 잡으면 $p_{n_2} = \tilde p_{k_2}$인 $n_2 > n_1$을 찾을 수 있다 (치역이 무한이므로 $n_1$보다 큰 인덱스를 무한히 많이 가진다). 귀납적으로 $n_1 < n_2 < \cdots$와 $k_1 < k_2 < \cdots$를 얻으며 $p_{n_i} = \tilde p_{k_i}$, $k_i \geq i$.

$\lim \tilde p_n = p$이고 $k_i \to \infty$이므로 $\lim_i \tilde p_{k_i} = p$. 따라서 $\lim p_{n_i} = p$.

□
English

Problem. If $\{\tilde p_n\} \subset \{p_n\}$ (as ranges), $\{\tilde p_n\}$ has infinite range and $\lim \tilde p_n = p$, show $\{p_n\}$ has a subsequence converging to $p$.

For each $\tilde p_k = p_{m_k}$. Since the range is infinite, we can pick increasing $k_1 < k_2 < \cdots$ and corresponding $n_1 < n_2 < \cdots$ with $p_{n_i} = \tilde p_{k_i}$. As $\tilde p_{k_i} \to p$, $p_{n_i} \to p$. $\square$

EX 3.16 점열적 콤팩트 ⇒ 닫힘·유계 → DEF 3.13
한국어

문제. $(X,d)$가 거리 공간, $K \subset X$가 점열적 콤팩트라 하자. $K$가 (a) 유계이고 (b) 닫혀있음을 증명하거나 반증하라.

풀이. 두 성질 모두 성립한다.

(a) 유계. 귀류법으로 $K$가 유계가 아니라고 하자. $x_0 \in K$를 고정. 임의의 $n \in \mathbb{N}$에 대해 $p_n \in K$를 $d(p_n, x_0) \geq n$이 되도록 택할 수 있다. $\{p_n\}$이 $K$ 안의 수열이므로 점열적 콤팩트성에 의해 부분수열 $\{p_{n_i}\}$가 존재하여 $\lim p_{n_i} = p \in K$. 그러면 $\{p_{n_i}\}$는 수렴이므로 유계 (THM 3.4(c))이다. 그러나 $d(p_{n_i}, x_0) \geq n_i \to \infty$, 즉 유계가 아니다. 모순.

(b) 닫힘. $p$가 $K$의 극한점이면 $p \in K$임을 보이면 된다. THM 3.6에 의해 $K$ 안의 수열 $\{p_n\}$이 존재하여 $p_n \neq p$이고 $\lim p_n = p$. $K$가 점열적 콤팩트이므로 부분수열 $\{p_{n_i}\}$가 존재하여 $\lim p_{n_i} = q \in K$. 그러나 부분수열도 $p$에 수렴 (상수가 부분수열로 유지), 극한의 유일성에 의해 $q = p$. 따라서 $p \in K$.

□
English

Problem. If $K$ is sequentially compact, prove (a) $K$ is bounded, (b) $K$ is closed.

(a) If not bounded, pick $p_n \in K$ with $d(p_n, x_0) \geq n$. A convergent subsequence must be bounded (THM 3.4(c)), contradicting unboundedness.

(b) If $p$ is a limit point of $K$, take $\{p_n\} \subset K$ with $p_n \to p$ (THM 3.6). Sequential compactness gives a subsequence $\to q \in K$; by uniqueness $q = p \in K$. $\square$

EX 3.17 $d(p_n, p) < 1/n$이면 $p_n \to p$ → LEM 3.18
한국어

문제. $(X,d)$가 거리 공간, $E \subset X$, $p \in X$라 하자. $E$ 안의 수열 $\{p_n\}$이 모든 $n \in \mathbb{N}$에 대해 $d(p_n, p) < 1/n$을 만족하면 $\lim p_n = p$임을 보여라.

풀이. $\varepsilon > 0$을 주자. 아르키메데스 성질에 의해 $1/N < \varepsilon$인 $N \in \mathbb{N}$이 존재.

모든 $n \geq N$에 대해

$$d(p_n, p) < \frac{1}{n} \leq \frac{1}{N} < \varepsilon.$$

따라서 $\lim p_n = p$.

□
English

Problem. If $d(p_n, p) < 1/n$ for all $n$, show $\lim p_n = p$.

Given $\varepsilon > 0$, pick $N$ with $1/N < \varepsilon$ (Archimedean). Then $d(p_n, p) < 1/n \leq 1/N < \varepsilon$ for $n \geq N$. $\square$

EX 3.21 임의의 $\varepsilon_1 \varepsilon_2$로 바꿀 수 있다 → DEF 3.20
한국어

문제. $(X,d)$가 거리 공간, $\{p_n\}$이 $p \in X$에 수렴한다고 하자. 모든 양의 상수 $\varepsilon_1, \varepsilon_2 > 0$에 대해 양의 정수 $N$이 존재하여 모든 $n \geq N$에 대해 $d(p_n, p) < \varepsilon_1 \varepsilon_2$임을 보여라.

풀이. $\varepsilon_1, \varepsilon_2 > 0$이 주어졌다. $\varepsilon := \varepsilon_1 \varepsilon_2 > 0$으로 놓자.

$\lim p_n = p$이므로 $\varepsilon > 0$에 대해 $N$이 존재하여 모든 $n \geq N$에 대해 $d(p_n, p) < \varepsilon = \varepsilon_1 \varepsilon_2$.

(핵심: 수렴 정의의 $\varepsilon$은 임의의 양의 실수이므로 형태가 무엇이든 양수이기만 하면 된다.)

□
English

Problem. If $\{p_n\}$ converges to $p$, show for any $\varepsilon_1, \varepsilon_2 > 0$ there exists $N$ with $d(p_n, p) < \varepsilon_1 \varepsilon_2$ for $n \geq N$.

Set $\varepsilon := \varepsilon_1 \varepsilon_2 > 0$. Apply the convergence definition with this $\varepsilon$. $\square$

EX 3.26 폐포의 포함 관계 → LEM 3.25
한국어

문제. $(X,d)$가 거리 공간, $E_1 \subset E_2 \subset X$라 하자. $\overline{E_1} \subset \overline{E_2}$임을 보여라.

풀이. $p \in \overline{E_1} = E_1 \cup E_1'$이라 하자.

Case 1. $p \in E_1 \subset E_2 \subset \overline{E_2}$.

Case 2. $p \in E_1'$, 즉 $p$는 $E_1$의 극한점. 임의의 근방 $N_\varepsilon(p)$은 $E_1 \setminus \{p\}$의 점을 포함. $E_1 \subset E_2$이므로 이 점은 $E_2 \setminus \{p\}$의 점이기도 하다. 따라서 $p$는 $E_2$의 극한점, 즉 $p \in E_2' \subset \overline{E_2}$.

어느 경우든 $p \in \overline{E_2}$.

□
English

Problem. If $E_1 \subset E_2 \subset X$, show $\overline{E_1} \subset \overline{E_2}$.

$p \in \overline{E_1} = E_1 \cup E_1'$. If $p \in E_1$, then $p \in E_2 \subset \overline{E_2}$. If $p \in E_1'$, every $N_\varepsilon(p)$ meets $E_1 \setminus \{p\} \subset E_2 \setminus \{p\}$, so $p \in E_2' \subset \overline{E_2}$. $\square$

EX 3.28 THM 3.27 증명에서 교집합은 한 점 → THM 3.27
한국어

문제. THM 3.27의 증명에서 구성한 $\overline{E_n} := \overline{\{p_n, p_{n+1}, \ldots\}}$에 대해 $\bigcap_{n=1}^\infty \overline{E_n} = \{p\}$ (정확히 한 점)임을 보여라.

풀이. THM 3.27 증명에서 $p \in \bigcap \overline{E_n}$이고 $\lim p_n = p$를 보였다. $p$의 유일성을 증명하자.

$p, p' \in \bigcap_{n=1}^\infty \overline{E_n}$이라 하자. $\varepsilon > 0$을 주자. $\{p_n\}$이 코시이므로 $N$이 존재하여 $n, m \geq N$이면 $d(p_n, p_m) < \varepsilon/3$.

$p \in \overline{E_N}$이므로 LEM 3.25에 의해 $p_{n_1} \in E_N$ ($n_1 \geq N$)이 존재하여 $d(p, p_{n_1}) < \varepsilon/3$. 마찬가지로 $p' \in \overline{E_N}$이므로 $p_{n_2} \in E_N$ ($n_2 \geq N$)이 존재하여 $d(p', p_{n_2}) < \varepsilon/3$.

삼각부등식에 의해

$$d(p, p') \leq d(p, p_{n_1}) + d(p_{n_1}, p_{n_2}) + d(p_{n_2}, p') < \frac{\varepsilon}{3} + \frac{\varepsilon}{3} + \frac{\varepsilon}{3} = \varepsilon.$$

$\varepsilon$이 임의이므로 $d(p, p') = 0$, 즉 $p = p'$.

□
English

Problem. Show that $\bigcap_{n=1}^\infty \overline{E_n} = \{p\}$ in the proof of THM 3.27.

THM 3.27 gives $p \in \bigcap \overline{E_n}$. If $p' \in \bigcap \overline{E_n}$ too, given $\varepsilon > 0$ find Cauchy $N$ with $d(p_n, p_m) < \varepsilon/3$ and use LEM 3.25 to get $p_{n_1}, p_{n_2} \in E_N$ with $d(p, p_{n_1}), d(p', p_{n_2}) < \varepsilon/3$. Triangle gives $d(p, p') < \varepsilon$. $\square$

EX 3.29 THM 3.14로 THM 3.27 증명 → THM 3.27
한국어

문제. THM 3.14(콤팩트 ⇒ 점열적 콤팩트)를 이용하여 THM 3.27을 증명하라.

증명. $X$가 콤팩트, $\{p_n\}$이 $X$ 안의 코시 수열이라 하자. THM 3.14에 의해 $X$는 점열적 콤팩트이므로 부분수열 $\{p_{n_i}\}$와 $p \in X$가 존재하여 $\lim_{i\to\infty} p_{n_i} = p$.

주장: $\lim_{n\to\infty} p_n = p$. $\varepsilon > 0$을 주자.

$\{p_n\}$이 코시이므로 $N_1$이 존재하여 $n, m \geq N_1$이면 $d(p_n, p_m) < \varepsilon/2$.

$\lim p_{n_i} = p$이므로 $I$가 존재하여 $i \geq I$이면 $d(p_{n_i}, p) < \varepsilon/2$.

$N := \max\{N_1, n_I\}$로 놓자. 모든 $n \geq N$에 대해, $i$를 $n_i \geq N$가 되도록 충분히 큰 $i \geq I$로 고르면 ($n_i \to \infty$이므로 가능),

$$d(p_n, p) \leq d(p_n, p_{n_i}) + d(p_{n_i}, p) < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon.$$
□
English

Problem. Prove THM 3.27 using THM 3.14.

By THM 3.14, $X$ is sequentially compact, so $\{p_n\}$ has a subsequence $p_{n_i} \to p$. Given $\varepsilon > 0$, Cauchy gives $N_1$ with $d(p_n,p_m) < \varepsilon/2$; convergence gives $I$ with $d(p_{n_i},p) < \varepsilon/2$. For $n \geq N_1$, pick $i \geq I$ with $n_i \geq N_1$; triangle gives $d(p_n, p) < \varepsilon$. $\square$

EX 3.31 COR 3.30의 증명 (두 방법) → COR 3.30
한국어

문제. COR 3.30을 (a) THM 3.27을 이용하여, (b) THM 3.27 없이 증명하라.

(a) THM 3.27 이용. $\{p_n\}$이 $\mathbb{R}^k$ 안의 코시 수열이라 하자. LEM 3.23에 의해 $\{p_n : n \in \mathbb{N}\}$은 유계이므로 $k$-cell $I$가 존재하여 $\{p_n\} \subset I$. $I$는 THM 2.76에 의해 콤팩트. $\{p_n\}$이 $I$ 안의 코시 수열이므로 THM 3.27에 의해 $\{p_n\}$은 수렴.

(b) THM 3.27 없이. LEM 3.23에 의해 $\{p_n\}$은 유계. COR 3.15(볼차노-바이어슈트라스)에 의해 수렴 부분수열 $\{p_{n_i}\}$가 존재, $\lim p_{n_i} = p \in \mathbb{R}^k$.

$\{p_n\}$이 코시이므로 임의의 $\varepsilon > 0$에 대해 $N_1$이 존재하여 $n, m \geq N_1$이면 $d(p_n, p_m) < \varepsilon/2$. 수렴으로 $I$가 존재하여 $i \geq I$이면 $d(p_{n_i}, p) < \varepsilon/2$. $n \geq N_1$에 대해 $n_i \geq N_1$인 $i \geq I$를 택하면

$$d(p_n, p) \leq d(p_n, p_{n_i}) + d(p_{n_i}, p) < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon.$$
□
English

Problem. Prove COR 3.30 (a) using THM 3.27, (b) without THM 3.27.

(a) By LEM 3.23 $\{p_n\}$ bounded, so in a $k$-cell $I$ (compact, THM 2.76). Apply THM 3.27 in $I$.

(b) By LEM 3.23 bounded; COR 3.15 gives a convergent subsequence $p_{n_i} \to p$. Combine with Cauchy via triangle (see EX 3.29). $\square$

EX 3.34 $\mathbb{Q}$는 완비가 아님 → DEF 3.32
한국어

문제. $\mathbb{Q}$를 표준 거리 $d(x,y) = |x - y|$로 생각한 거리 공간이라 하자. $\mathbb{Q}$가 완비가 아님을 보여라.

풀이. $\sqrt{2}$의 십진 근사를 사용한다. $a_1 := 1, a_2 := 1.4, a_3 := 1.41, a_4 := 1.414, \ldots$로 $a_n$을 $\sqrt{2}$의 소수 $n-1$자리 절삭으로 정의하면 각 $a_n \in \mathbb{Q}$이다.

$\{a_n\}$은 코시. $|a_n - a_m| \leq 10^{-(\min\{n,m\}-1)} \to 0$ ($n,m \to \infty$).

$\{a_n\}$은 $\mathbb{Q}$에서 수렴하지 않는다. $\mathbb{R}$에서는 $a_n \to \sqrt{2}$. $\{a_n\}$이 $\mathbb{Q}$에서 어떤 $q$에 수렴한다면 $\mathbb{R}$에서도 수렴하므로 극한의 유일성에 의해 $q = \sqrt{2}$. 그러나 $\sqrt{2} \notin \mathbb{Q}$, 모순.

따라서 $\mathbb{Q}$ 안의 코시 수열로 $\mathbb{Q}$에서 수렴하지 않는 것이 존재하므로 $\mathbb{Q}$는 완비가 아니다.

□
English

Problem. Let $\mathbb{Q}$ be the rationals with $d(x,y) = |x-y|$. Show $\mathbb{Q}$ is not complete.

Decimal truncations $a_n$ of $\sqrt{2}$ are rational, Cauchy (since $|a_n - a_m| \leq 10^{-\min\{n,m\}+1}$), and converge to $\sqrt{2}$ in $\mathbb{R}$. Any $\mathbb{Q}$-limit would equal $\sqrt{2} \notin \mathbb{Q}$. $\square$

EX 3.38 LEM 3.37(b)의 증명 → LEM 3.37
한국어

문제. LEM 3.37(b)를 증명하라: $A \subset \mathbb{R}$, $A \neq \emptyset$이고 $\inf A \in \mathbb{R}$이면 임의의 $\varepsilon > 0$에 대해 $b \in A$가 존재하여 $b < \inf A + \varepsilon$.

증명. $\varepsilon > 0$을 주자. $b \in A$가 $b < \inf A + \varepsilon$을 만족하는 것이 없다고 하자. 그러면 모든 $b \in A$에 대해 $b \geq \inf A + \varepsilon$.

이는 $\inf A + \varepsilon$이 $A$의 하계라는 뜻. $\inf A$는 최대 하계(greatest lower bound)이므로

$$\inf A \geq \inf A + \varepsilon,$$

즉 $0 \geq \varepsilon$. 그러나 $\varepsilon > 0$이므로 모순.

□
English

Problem. Prove LEM 3.37(b).

Suppose no such $b$; then $b \geq \inf A + \varepsilon$ for all $b \in A$, so $\inf A + \varepsilon$ is a lower bound. Since $\inf A$ is greatest, $\inf A \geq \inf A + \varepsilon$, contradicting $\varepsilon > 0$. $\square$

EX 3.41 상극한 = 하극한이면 수렴 (Difficult!) → COR 3.40
한국어

문제. $\{a_n\}$이 $\mathbb{R}$ 안의 수열이고 $\limsup a_n = \liminf a_n \in \mathbb{R}$이면 $\{a_n\}$이 수렴함을 보여라.

증명. $L := \limsup a_n = \liminf a_n$으로 놓자. $\lim a_n = L$을 보이면 된다.

$\varepsilon > 0$을 주자. $\limsup$과 $\liminf$의 정의에서:

$$L = \inf_{k \geq 1}\{\sup_{n \geq k} a_n\}.$$

LEM 3.37(b)에 의해 $K_1 \in \mathbb{N}$이 존재하여 $\sup_{n \geq K_1} a_n < L + \varepsilon$. 특히 모든 $n \geq K_1$에 대해 $a_n < L + \varepsilon$.

유사하게 $L = \sup_{k \geq 1}\{\inf_{n \geq k} a_n\}$, LEM 3.37(a)에 의해 $K_2 \in \mathbb{N}$이 존재하여 $\inf_{n \geq K_2} a_n > L - \varepsilon$. 특히 모든 $n \geq K_2$에 대해 $a_n > L - \varepsilon$.

$N := \max\{K_1, K_2\}$로 놓으면 모든 $n \geq N$에 대해

$$L - \varepsilon < a_n < L + \varepsilon, \quad \text{즉} \quad |a_n - L| < \varepsilon.$$

따라서 $\lim a_n = L$.

□
English

Problem. If $\limsup a_n = \liminf a_n \in \mathbb{R}$, show $\{a_n\}$ converges.

Let $L$ be the common value. Given $\varepsilon > 0$, LEM 3.37 gives $K_1$ with $\sup_{n \geq K_1} a_n < L + \varepsilon$ and $K_2$ with $\inf_{n \geq K_2} a_n > L - \varepsilon$. For $n \geq \max\{K_1,K_2\}$, $|a_n - L| < \varepsilon$. $\square$

EX 3.42 THM 3.39(b)의 증명 → THM 3.39
한국어

문제. THM 3.39(b)를 증명하라: $\liminf a_n \in \mathbb{R}$이면 부분수열 $\{a_{n_k}\}$가 존재하여 $\lim a_{n_k} = \liminf a_n$.

증명. $L := \liminf a_n = \sup_{k \geq 1}\{\inf_{n \geq k} a_n\}$이라 하자.

LEM 3.37(a)에 의해 $K_1 \in \mathbb{N}$이 존재하여 $L - 1 < \inf_{n \geq K_1} a_n$. LEM 3.37(b)에 의해 $n_1 \geq K_1$이 존재하여 $a_{n_1} < \inf_{n \geq K_1} a_n + 1$. 즉

$$L - 1 < a_{n_1} < L + 1.$$

유사하게 $\tilde K_2$가 존재하여 $L - 1/2 < \inf_{n \geq \tilde K_2} a_n$. $K_2 := \max\{\tilde K_2, n_1\} + 1$으로 놓으면 $\inf_{n \geq K_2} a_n \geq \inf_{n \geq \tilde K_2} a_n > L - 1/2$이고 $K_2 > n_1$. LEM 3.37(b)로 $n_2 \geq K_2$가 존재하여 $a_{n_2} < \inf_{n \geq K_2} a_n + 1/2 \leq L + 1/2$. 따라서 $|a_{n_2} - L| \leq 1/2$, $n_2 > n_1$.

귀납적으로 $n_1 < n_2 < \cdots$가 존재하여 $|a_{n_k} - L| \leq 1/k$. $1/k \to 0$이므로 $\lim a_{n_k} = L$.

□
English

Problem. Prove THM 3.39(b).

Symmetric to (a). Set $L = \liminf a_n$. Using LEM 3.37 to $\sup_k \{\inf_{n \geq k} a_n\}$, build $n_1 < n_2 < \cdots$ with $|a_{n_k} - L| \leq 1/k$. $\square$

EX 3.44 단조 수열의 수렴 → DEF 3.43
한국어

문제. $\{a_n\}$이 $\mathbb{R}$ 안의 단조 수열이라 하자.

(1) $\{a_n : n \in \mathbb{N}\}$이 유계이면 $\{a_n\}$은 $\mathbb{R}$에서 극한을 가짐을 보여라.

(2) 일반적으로 $\{a_n\}$이 확장 실수계 $\overline{\mathbb{R}}$에서 극한을 가지는가? 증명하거나 반증하라.

(1) 증명. 단조증가로 가정 (감소는 대칭). $A := \{a_n : n \in \mathbb{N}\}$은 유계, $a := \sup A \in \mathbb{R}$.

$\varepsilon > 0$을 주자. LEM 3.37(a)에 의해 $a_N \in A$가 존재하여 $a - \varepsilon < a_N$. $\{a_n\}$이 단조증가이므로 모든 $n \geq N$에 대해 $a_n \geq a_N > a - \varepsilon$. 또한 $a_n \leq a$. 따라서 $|a_n - a| < \varepsilon$.

$\lim a_n = a$.

(2) 증명. 참. 단조증가 $\{a_n\}$에 대해:

• $\{a_n\}$이 유계이면 (1)에 의해 $\mathbb{R}$에서 수렴.

• $\{a_n\}$이 유계가 아니면 ($\sup A = \infty$), 임의의 $M > 0$에 대해 $a_N > M$인 $N$이 존재. 단조증가성에 의해 $n \geq N$이면 $a_n \geq a_N > M$. 즉 $\lim a_n = +\infty$.

단조감소의 경우도 대칭. 어느 경우든 $\{a_n\}$은 $\overline{\mathbb{R}}$에서 극한을 가진다.

□
English

Problem. Let $\{a_n\}$ be monotonic. (1) If bounded, show a limit exists in $\mathbb{R}$. (2) Does a limit exist in $\overline{\mathbb{R}}$?

(1) Assume increasing. $a = \sup\{a_n\} \in \mathbb{R}$. Given $\varepsilon$, LEM 3.37 gives $a_N > a - \varepsilon$; monotonicity forces $|a_n - a| < \varepsilon$ for $n \geq N$.

(2) Yes. If unbounded and increasing, $a_n \to +\infty$: for $M$, find $N$ with $a_N > M$; monotonicity gives $a_n > M$ for $n \geq N$. Symmetric for decreasing. $\square$

EX 3.46 꼬리 급수의 동치 → DEF 3.45
한국어

문제. $\{a_n\}$이 $\mathbb{R}^k$ 안의 수열이라 하자.

(1) $\sum_{n=1}^\infty a_n = s$일 필요충분조건은 임의의 $k \in \mathbb{N}$에 대해 $\sum_{n=k}^\infty a_n = s - \sum_{n=1}^{k-1} a_n$임을 보여라.

(2) $\sum a_n = s$이 "어떤 $k$에 대해 $\sum_{n=k}^\infty a_n = s - \sum_{n=1}^{k-1} a_n$"과 동치인가? 증명하거나 반증하라.

(1) 증명. $s_n := \sum_{i=1}^n a_i$로 부분합을 놓자.

($\Rightarrow$) $\sum_{n=1}^\infty a_n = s$이면 $\lim s_n = s$. 임의 $k \in \mathbb{N}$에 대해 $\tilde s_n := s_{k-1+n} - s_{k-1} = \sum_{i=k}^{k-1+n} a_i$. $\lim s_{k-1+n} = s$이므로

$$\lim \tilde s_n = s - s_{k-1} = s - \sum_{n=1}^{k-1} a_n.$$

따라서 $\sum_{n=k}^\infty a_n = s - \sum_{n=1}^{k-1} a_n$.

($\Leftarrow$) 특히 $k = 1$인 경우 $\sum_{n=1}^\infty a_n = s - 0 = s$.

(2) 증명 (참). $\sum_{n=k}^\infty a_n = s - \sum_{n=1}^{k-1} a_n$ ($=: t_k$)이 어떤 $k$에 대해 성립한다고 하자. 즉 $\lim_{n \to \infty}(s_{k-1+n} - s_{k-1}) = t_k$. 그러면 $\lim s_{k-1+n} = t_k + s_{k-1} = s$. 수열 $\{s_m\}$은 $m = k-1+n$ ($n \to \infty$)일 때 $s$로 수렴. $m \to \infty$ 일 때 $n \to \infty$이므로 $\lim_{m\to\infty} s_m = s$, 즉 $\sum_{n=1}^\infty a_n = s$.

□
English

Problem. (1) Show $\sum_{n=1}^\infty a_n = s$ iff $\sum_{n=k}^\infty a_n = s - \sum_{n=1}^{k-1} a_n$ for all $k$. (2) Does one such $k$ suffice?

(1) $\sum_{n=k}^\infty a_n = \lim_n (s_{k-1+n} - s_{k-1})$; this equals $s - s_{k-1}$ iff $\lim s_{k-1+n} = s$ iff $\lim s_m = s$.

(2) Yes — same argument with one $k$ gives $\lim s_m = s$. $\square$

EX 3.48 부분합의 코시성 → THM 3.47
한국어

문제. $\{a_n\}$이 $\mathbb{R}^k$ 안의 수열, $\{s_n\}$을 부분합이라 하자. $\{s_n\}$이 $\mathbb{R}^k$ 안의 코시 수열일 필요충분조건은 임의의 $\varepsilon > 0$에 대해 $N$이 존재하여 모든 $m \geq n \geq N$에 대해 $\left|\sum_{k=n}^m a_k\right| < \varepsilon$임을 보여라.

증명. $\{s_n\}$이 코시 $\iff$ $\forall \varepsilon > 0, \exists N, \forall n_1, n_2 \geq N$: $|s_{n_1} - s_{n_2}| < \varepsilon$.

WLOG $n_1 \geq n_2$로 놓자. $n_1 = n_2$이면 $|s_{n_1} - s_{n_2}| = 0 < \varepsilon$이므로 자명. $n_1 > n_2$이면 $n := n_2 + 1$, $m := n_1$로 놓으면 $m \geq n \geq N + 1$이고

$$s_{n_1} - s_{n_2} = \sum_{k=n_2+1}^{n_1} a_k = \sum_{k=n}^m a_k.$$

($\Rightarrow$) $\{s_n\}$ 코시 $\Rightarrow$ $\varepsilon > 0$에 $\tilde N$이 존재하여 $m \geq n \geq \tilde N + 1$이면 $|\sum_{k=n}^m a_k| < \varepsilon$. $N := \tilde N + 1$로 놓자. 모든 $m \geq n \geq N$에 대해 $|\sum_{k=n}^m a_k| < \varepsilon$ ($m = n$이면 $|a_n|$이 작은가? 사실 $s_n - s_{n-1} = a_n$이므로 $n-1 \geq \tilde N$이어야 함; 항상 $N$을 충분히 큰 값으로 잡으면 됨).

($\Leftarrow$) 역방향: 조건이 성립하면 $n_1 > n_2 \geq N - 1$에서 $|s_{n_1} - s_{n_2}| = |\sum_{k=n_2+1}^{n_1} a_k| < \varepsilon$. ($N' := N$로 놓으면 $n_1, n_2 \geq N'$에서 $|s_{n_1} - s_{n_2}| < \varepsilon$.)

□
English

Problem. Show $\{s_n\}$ is Cauchy iff $\forall \varepsilon > 0$ $\exists N$: $m \geq n \geq N \implies |\sum_{k=n}^m a_k| < \varepsilon$.

For $n_1 > n_2$, $s_{n_1} - s_{n_2} = \sum_{k=n_2+1}^{n_1} a_k$. Equivalence follows by adjusting the index $N$ appropriately. $\square$

EX 3.50 $\left|\sum a_n\right| \leq \sum |a_n|$ (Difficult!) → COR 3.49
한국어

문제. $\{a_n\}$이 $\mathbb{R}^k$ 안의 수열이고 $\sum_{n=1}^\infty a_n$이 절대수렴한다고 하자. 임의의 $m \in \mathbb{N}$에 대해

$$\left| \sum_{n=m}^\infty a_n \right| \leq \sum_{n=m}^\infty |a_n|$$

임을 보여라.

증명. $\sum a_n$이 절대수렴하므로 COR 3.49에 의해 수렴. EX 3.46(1)에 의해 $\sum_{n=m}^\infty a_n$도 수렴한다.

$s_N := \sum_{n=m}^{m+N} a_n$, $t_N := \sum_{n=m}^{m+N} |a_n|$이라 놓자. 유한합 삼각부등식에 의해 모든 $N$에 대해

$$|s_N| \leq t_N.$$

$N \to \infty$로 극한을 취하면:

• $s_N \to \sum_{n=m}^\infty a_n$ ($\mathbb{R}^k$에서)

• 절댓값 함수 $|\cdot|: \mathbb{R}^k \to \mathbb{R}$은 연속 (왜냐하면 $||x| - |y|| \leq |x - y|$이므로 립시츠). 따라서 $|s_N| \to \left|\sum_{n=m}^\infty a_n\right|$.

• $t_N \to \sum_{n=m}^\infty |a_n|$ (단조증가 수열의 유계 극한).

$\mathbb{R}$에서 부등식 $|s_N| \leq t_N$의 양변에 극한을 취하면 (비교는 극한을 보존) $\left|\sum_{n=m}^\infty a_n\right| \leq \sum_{n=m}^\infty |a_n|$.

□
English

Problem. If $\sum a_n$ absolutely converges, show $\left|\sum_{n=m}^\infty a_n\right| \leq \sum_{n=m}^\infty |a_n|$ for any $m \in \mathbb{N}$.

Finite triangle: $|\sum_{n=m}^{m+N} a_n| \leq \sum_{n=m}^{m+N} |a_n|$. Take $N \to \infty$; by continuity of $|\cdot|$ and monotone convergence of tails, both sides have limits and the inequality persists. $\square$
